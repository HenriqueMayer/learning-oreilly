## DevOps to MLOps Bootcamp: Build & Deploy ML Systems End-to-End
Link: https://learning.oreilly.com/course/devops-to-mlops/9781806118694/

### Overview
- Design and automate complete ML workflows by applying MLOps principles
- Manage model experiments and tracking using tools like MLFlow and Jupyter
- Build and containerize deployable ML systems using FastAPI and Docker
- Automate CI/CD pipelines integrating GitHub Actions and DockerHub
- Deploy, scale, and monitor infrastructure using Kubernetes and Prometheus
